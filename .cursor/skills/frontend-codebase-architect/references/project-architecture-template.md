## 16. 新增接口步骤

1. 明确接口所属业务域与数据模型。
2. 在 API 模块新增请求方法与类型定义。
3. 复用统一拦截器与错误处理流程。
4. 在页面/状态层接入并处理异常分支。
5. 更新文档中的调用约定与返回结构说明。

3. 明确是否持久化及其清理策略。
4. 在消费侧按需订阅，避免冗余渲染。
